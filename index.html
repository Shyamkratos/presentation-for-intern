Deploy: GitHub → Render

  1. Create a new GitHub repo and add this HTML file as index.html. Add an /assets/ folder with the PDFs and any images.
  2. Commit & push: git add .; git commit -m "Add site"; git push
  3. On Render: Create a new Static Site, connect your GitHub repo and set the build command to: none and publish directory to /.
  4. Optional: Use a custom domain, add HTTPS, and enable automatic deploys from your main branch.
